Description

 

A brief description of how the Application runs its features and options are listed below.

Sample Instructor

User ID : SYSTEM Password : 123456

Sample Student

User ID : ssi Password : 123456

The first form that pops up is the Login Form it asks of a UserID and a Password.
There a two modes in which a user can log in, An Instructor Mode or A Student Mode.

Instructor Mode

  • Assuming that the Instructor has entered a valid UserID and Password, the next form that appears is the Test Creator Form.
  • To open a Question Bank file, use File >Open Question Bank thus all the Questions get loaded into the List Box on the right hand side.
  • To create a new file, use File>New Question Bank this will invoke the Question Bank Editor Form or use Edit Question Bank Button.
  • Once in the Question Bank Editor Form Instructor can Add, Delete, Update and Save Questions to a Data base file(.mdb).

The Test Creator Form has a number options for creating the Test Paper.

  • Creating Test Paper.
    Go to File>New Test Paper, similarly Instructor can open an existing Test Paper File>Open Test Paper.
  • Time Options.
    Instructor can choose either Unlimited or Limited Time, for Limited Time option the Instructor has to give the number of seconds moreover he can give different Time (number of seconds) for different Questions.
  • Mark Options.
    Positive value is the Marks awarded for a correct answer.
    Negative value is the Marks deducted for an incorrect answer (its optional).
  • Pass Percentage.
    Using the scroll bar one can specify the percentage the Student has to score to Pass the particular Test.

 

Once the Test Paper has been created, the Instructor now needs to Assign this test to Students using Test Assignor.

  • Assign Test.
    Select a Student from the list box on the left hand side and then clicking Assign Test Button will assign the test to that particular Student.
  • Expire Date.
    Each Test is assigned with an Expire Date, it is the time limit within which the student must Attempt the test.

 

Instructors can create New Student Logins using User ID manager.

 

  • New Student Login.
    Using User ID manager form Instructors can Add new Students, by simply entering the required fields and clicking Create Account button.
  • New Instructor Login.
    Similarly New Instructor can also be created from the Login menu Login>New Instructor login .
  • Changing Password.
    An Instructor can change password of any Student Login from the menu Login>Edit Student Login, but he cannot edit other Instructor logins.



Student Mode


If the user has a valid Student ID and Password then he logs in as a Student.
All the test that have been assigned to him appear in combo box and by selecting the name and clicking the Start Test button the Student can start the test.

 

 

 

The Test Paper Form gives all the Information to about the test to the User

  • The question appears in the text box with the four options given below
    the Student has to choose any one Option and the click Next Question Button.
  • If the Student fails to answer the Question in the given time then the next Question gets loaded up.
  • If the Student chooses NO ATTEMPT option then no negative marks will be deducted from his total.
  • Abort Test will abort the test marking all the answers as NO ATTEMPT.


Reports

Reports are summarized information in a well-formatted manner meant for higher managerial levels to help them in Decision-making.

The current Application creates a total of four such reports.
1. Test Result Report.
2. Student Report.
3. Instructor Report.
4. Test Report.

  1. Test Result Report.

    2. This Report is created as soon as the student completes the Test, it gives information about what question did the Student attempt and the Positive or Negative marks scored and did the Student Pass of Fail In the Test.

  2. Student Report.
    This report gives the over all Performance of a student, how many test did he attempt, date when he took the test, what percentage he scored in each and finally his Average Score. This Report can be generated from the Test Creator Report menu Report>Student Report.
  3. Instructor Report.
    This Report give the details of a particular Instructor, the total number of test he created, the average score of the student in each test. This Report can be generated from the Test Creator Report menu Report>Instructor Report.
  4. Test Report.
    This Report gives the details of a particular Test, the total number of students who took the test, the average score of the all student. This Report can be generated from the Test Creator Report menu Report>Test Report.
